《『 ( こうげき ) 』 ( ふう ) じ/Block Attack》 

通常魔法
 相手フィールド上に表側攻撃表示で存在するモンスター１体を選択して表側守備表示にする。

Vol.7で登場した通常魔法。 相手モンスター１体を守備表示にする効果を持つ。

相手モンスターの表示形式を変更するだけなので、単体ではアドバンテージを得ることができない。 攻撃力が高く、守備力が低いモンスターに対して発動し、戦闘破壊を補助するために使用することになる。

ただし、速攻魔法の《エネミーコントローラー》等の上位互換が多く存在する為、優先度は低い。 守備表示が存在しないリンクモンスターの登場も向かい風であり、ファンデッキ以外での採用は厳しい。

  • 第１期の頃は、《ヂェミナイ・エルフ》や《デーモンの召喚》等の攻撃力が高く守備力が低いモンスターが主力だったため、時折採用されていた。
  • カード名に『』（二重かぎかっこ）が入っている珍しいカード。 長らくこのカードと《『守備』封じ》しか存在しなかったのだが、17年の時を経て《星遺物−『星杯』》が登場した。
  • 原作・アニメにおいて― アニメオリジナル「乃亜編」の「本田＆御伽＆静香vsビッグ４（大田）」戦で本田が使用。 ルールを知らない静香が攻撃表示で召喚した《慈悲深き修道女》を守備表示にしようとしたが、大田の《フォースフィールド》で無効化された。 同じく乃亜編の「闇遊戯＆城之内vsビッグ５」では城之内が使用。 《お注射天使リリー》を守備表示に変更させることで、自己強化効果を間接的に封じて戦闘破壊に繋いだ。
  • 海馬もこのカードを所有しており、アニメの第１話や「戦いの儀」では彼のジュラルミンケースの中に確認できる。
  • 漫画ARC-Vの「遊矢（ユート・ユーリ）vs零児」戦では、零児が使用。 この時は、イラストはそのままでカード名が《攻撃封じ》となっている。 《ダーク・アンセリオン・ドラゴン》を守備表示に変更して、永続罠《 ( ダブルオー ) ラッシュ》による連続攻撃へ繋いだ。 その後、ＤＤＤのペンデュラム召喚をトリガーに『封じ』カードをサルベージする永続魔法《無限封印》で再利用し、今度は《スターヴ・ヴェネミー・ドラゴン》を守備表示に変更して同様のコンボに繋いだ。
  • 対となる《『守備』封じ》と異なり、OCGで登場した後にアニメで採用されたカードである。
  • アニメSEVENS等ではラッシュデュエル版のこのカードが登場している。 RUSH:《『攻撃』封じ》を参照。
  • コナミのゲーム作品において― 真DM2では、通常罠として登場。 相手モンスターの攻撃を無効にして、そのまま行動終了にする効果を持つ。
  • 遊戯王デュエルモンスターズ 最強カードバトル！では、魔法カードのチュートリアルに使われる他、初期デッキに複数投入されている。 使用時には主人公の専用ボイスも用意されている。
  • デュエルリンクスでは初期デッキに複数投入されている。 本田には専用ボイスも用意されている。
  • その他の商品展開― 遊戯王ラッシュデュエルでも同一の効果で実装されている。 RUSH:《『攻撃』封じ》を参照。
